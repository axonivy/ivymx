Annotations and utility classes to register POJO's as JMX {@link com.axonivy.jmx.MBean MBeans}. To start have a look at the class {@link com.axonivy.jmx.MBeans} and the annotation {@link com.axonivy.jmx.MBean}.

Simple Example:

 {@code @MBean}("GettingStarted:name=Person")
 public class Person
 {
   public static void main(String[] args)
   {
     Person person = new Person();
     MBeans.registerMBeanFor(person);
   }
 }
